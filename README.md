# CS370
In this project, I developed an intelligent pirate agent capable of navigating a maze to locate a treasure using a Deep Q-Learning algorithm. The goal was to train the agent to make optimal decisions through reinforcement learning by interacting with the environment and learning from its experiences.
The starter code provided the maze environment, neural network architecture, experience replay class, and portions of the training function. My primary contribution was completing the Deep Q-Learning training loop by implementing the agent's decision-making process, experience replay, model training, target network updates, and exploration versus exploitation strategy using an epsilon-greedy policy. I also integrated the logic necessary to track training progress and determine when the model had successfully learned to solve the maze.
